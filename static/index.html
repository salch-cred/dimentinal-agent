Core Safety Pillars

Four non-negotiable guarantees.

We separate the generation of plans from the mathematical execution. A sandboxed core decides if actions are dimensionally sound.

01 / SOURCE

Cell-level Trace

Every single operand is mapped to its precise spreadsheet cell coordinate. No detached facts.

02 / DIMENSIONS

Dimensional Safety

Entities, flows, scales, and time periods must align perfectly. Adding scale mismatch or RAG mismatch is blocked.

03 / ARITHMETIC

Deterministic Core

Calculations run outside the LLM in a rigid execution kernel. Never let a model calculate digits.

04 / AUDITED

Adversarial Auditor

A separate auditor agent attempts to dispute the calculated values against other document contexts.
